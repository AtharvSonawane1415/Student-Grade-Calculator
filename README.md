# Grade Calculator

A simple Python program that calculates and displays letter grades based on numerical marks input. This program includes input validation and provides personalized feedback messages.

## Features

- **Input Validation**: Ensures names contain valid characters and marks are within 0-100 range
- **Grade Calculation**: Automatically determines letter grades based on standard grading scale
- **Personalized Feedback**: Provides encouraging messages based on performance
- **User-Friendly Interface**: Clear prompts and formatted output display

## Grading Scale

| Marks Range | Grade | Message |
|-------------|-------|---------|
| 90-100 | A | Excellent! Outstanding performance! |
| 80-89 | B | Very Good! Keep it up! |
| 70-79 | C | Good work! Keep practicing! |
| 60-69 | D | Satisfactory! Can do better! |
| 0-59 | F | Need improvement! Work harder! |

## Requirements

- Python 3.x
- No external libraries required
